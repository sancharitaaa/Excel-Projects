# Excel Projects

A collection of financial analysis and data validation projects built in
Excel, covering ratio analysis, regression/risk-return modeling, and data
classification.

## Projects

### 1. Employee Equality Score Classification (`equality table.xlsx`)
Built as part of the Deloitte Data Analytics Job Simulation. Used Excel
functions (nested logical formulas) to validate and classify employee
equality scores into Fair, Unfair, and Highly Discriminative categories,
ensuring data accuracy for downstream reporting.

### 2. Quantitative Analysis — Lloyd Metals & Energy (`quantitative analysis lloyd metals.xlsx`)
Compared financial performance and valuation metrics against industry
peers. Conducted ratio and trend analysis using financial statement data to
assess profitability and investment potential.

### 3. Ratio Analysis — HCL Tech (`ratio analysis hcltech.xlsx`)
Calculated and analyzed key financial ratios from HCL Tech's financial
statements to evaluate profitability, efficiency, and overall financial
health.

### 4. Regression Analysis — HCL Tech vs. NIFTY (`regression analysis hcl tech.xlsx`)
Evaluated HCL Tech's stock risk and return behavior using regression
analysis and benchmark comparison against the NIFTY index. Built
scatterplots and Excel-based regression models to analyze correlation
between HCL Tech's stock movements and the broader market.

## Skills demonstrated

Advanced Excel functions, nested logical formulas, financial ratio
analysis, regression modeling, scatterplot visualization, benchmark/market
comparison analysis.

## Notes

These were built as academic/personal projects to practice financial
analysis techniques in Excel — not affiliated with or using data from any
company in a professional capacity (aside from the Deloitte simulation,
which used simulation-provided sample data).
